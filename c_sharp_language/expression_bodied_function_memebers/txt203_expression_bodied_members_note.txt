Expression bodied members can be public, protected, internal, private

Expression bodied members can be declared virtual or abstract

Expression bodied members can even override of base class method.

Expression bodied members can also be static.

Expression bodied memers require only one statement as expression. So block if code is not applicable with it.

Expression bodied members don't support loop (for, while, do while, etc) but they can be replaced with LINQ

Expression bodied members doesn't support conditional statement(if-else, nested if-else, switch etc) but if-else
can be replaced with ternary operator.
